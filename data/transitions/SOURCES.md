# Transitions

Short cross-fade clips played between channel changes. Not part of the editorial library — these are UI assets, not radio content.

All clips are normalized to the same standard as the rest of the library
(mono MP3 128 kbps, 44.1 kHz, -16 LUFS).

To swap or extend the set, drop new normalized MP3s into this directory.
The player picks one at random per channel change.
